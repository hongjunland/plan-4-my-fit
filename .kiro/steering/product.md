# 헬스 루틴 플래너 - MVP 제품 개요

## 📱 제품 설명

AI가 개인 맞춤형 헬스 루틴을 생성해주는 모바일 웹앱입니다. 사용자의 체력 수준, 목표, 선호도를 분석하여 최적화된 운동 계획을 제공하고, 진행 상황을 추적할 수 있습니다.

---

## 🎯 핵심 기능

### 1. 개인 맞춤 루틴 생성
- **AI 기반 루틴 생성**: GPT-4를 활용한 개인화된 운동 계획
- **다양한 분할 방식**: 전신, 상/하체, Push/Pull/Legs
- **목표별 최적화**: 근력 증가, 체중 감량, 체력 향상, 근육 증가

### 2. 스마트 루틴 관리
- **다중 루틴 저장**: 여러 루틴 생성 및 관리
- **활성화 시스템**: 한 번에 하나의 루틴만 활성화
- **루틴 편집**: 운동 수정, 복제, 삭제 기능

### 3. 직관적인 캘린더
- **3가지 뷰**: 오늘/주간/월간 캘린더
- **실시간 진행률**: 운동 완료 상태 추적
- **체크리스트**: 간편한 운동 완료 체크

### 4. 진행 상황 분석
- **주간/월간 통계**: 운동 완료율 시각화
- **연속 운동 기록**: 동기부여를 위한 스트릭 추적
- **목표 달성률**: 전체 루틴 진행 상황

---

## 👥 타겟 사용자

### 주요 타겟
- **헬스 초보자** (25-35세): 체계적인 운동 계획이 필요한 사용자
- **바쁜 직장인** (28-40세): 효율적인 운동 루틴을 원하는 사용자
- **홈트족** (20-45세): 집에서 운동하는 사용자

### 사용자 페르소나
```
👤 김민수 (29세, 직장인)
- 헬스장 등록했지만 뭘 해야 할지 모름
- 체계적인 운동 계획이 필요
- 모바일로 간편하게 관리하고 싶음

👤 이지영 (32세, 워킹맘)  
- 시간이 부족해서 효율적인 운동 필요
- 집에서도 할 수 있는 루틴 선호
- 진행 상황을 한눈에 보고 싶음
```

## 💡 핵심 가치 제안

### 🤖 AI 맞춤화
- "개인의 체력과 목표에 맞는 완벽한 루틴"
- 복잡한 운동 계획을 AI가 자동으로 생성

### 📱 간편함
- "3분만에 완성되는 나만의 운동 계획"
- 복잡한 설정 없이 바로 시작 가능

### 📊 동기부여
- "성취감을 주는 진행률 추적"
- 시각적 피드백으로 지속적인 동기 제공

## 🚀 MVP 범위

### ✅ 포함 기능
- 구글 소셜 로그인
- 프로필 설정 (키, 몸무게, 목표, 체력 수준)
- AI 루틴 생성 (GPT-4 기반)
- 루틴 관리 (생성, 활성화, 편집, 삭제)
- 캘린더 뷰 (오늘/주간/월간)
- 운동 체크리스트
- 기본 진행률 추적

### ❌ 제외 기능 (v2 예정)
- 구글 캘린더 연동
- 소셜 기능 (친구, 공유)
- 고급 분석 (운동량, 칼로리)
- 푸시 알림
- 운동 동영상/이미지

## 🎨 디자인 철학

### 토스 스타일 UI
- **색상**: #3182f6 (토스 블루) 메인 컬러
- **둥근 모서리**: 16px, 24px 라운드
- **그림자**: 미묘한 shadow-sm
- **타이포그래피**: 깔끔하고 읽기 쉬운 폰트

### 모바일 퍼스트
- **반응형**: 모바일 최적화 (max-width: 448px)
- **터치 친화적**: 최소 48px 터치 영역
- **직관적 네비게이션**: 하단 4개 탭

## 📈 성공 지표 (KPI)

### 사용자 참여
- **DAU/MAU**: 일간/월간 활성 사용자
- **루틴 완료율**: 생성된 루틴의 완료 비율
- **세션 시간**: 평균 앱 사용 시간

### 제품 성능
- **루틴 생성률**: 가입 후 루틴 생성 비율
- **재방문율**: 7일/30일 재방문 비율
- **운동 체크율**: 일일 운동 완료 체크 비율

### 비즈니스
- **사용자 획득 비용** (CAC)
- **사용자 생애 가치** (LTV)
- **월간 반복 수익** (MRR) - 유료 전환 시
