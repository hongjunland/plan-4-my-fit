# 🏋️ 헬스 루틴 플래너 PRD 개요

## 한 줄 요약

**"AI가 내 체력과 목표에 맞는 운동 루틴을 만들고, 캘린더로 관리하는 모바일 웹앱"**

---

## 🎯 문제 정의

### 현재 상황
- 헬스장에 가도 **뭘 해야 할지 모르겠다**
- 인터넷에서 찾은 루틴은 **내 체력에 안 맞는다**
- 부상이나 장비 제한을 **반영하기 어렵다**
- 운동 계획을 **체계적으로 추적하기 어렵다**

### 해결책
AI 맞춤 루틴 생성 + 캘린더 기반 운동 관리

---

## 👤 타겟 사용자

| 페르소나 | 특징 |
|---------|------|
| 헬스 입문자 | 운동을 시작하고 싶지만 뭘 해야 할지 모르는 사람 |
| 바쁜 직장인 | 제한된 시간 내에 효율적으로 운동하고 싶은 사람 |
| 부상 경험자 | 특정 부위를 피해서 운동해야 하는 사람 |

---

## ✨ 핵심 기능

### 1. 구글 원클릭 로그인
- 별도 회원가입 없이 구글 계정으로 바로 시작

### 2. AI 맞춤 루틴 생성 🤖
```
입력:
├── 신체 정보: 키, 몸무게
├── 운동 목적: 근력↑ / 체중↓ / 체력↑ / 근육량↑
├── 체력 수준: 입문자 / 초급자 / 중급자 / 상급자
├── 기간: 2주 ~ 12주
├── 주당 횟수: 2회 ~ 6회
└── 추가 요청: "어깨 부상", "덤벨만 있어요" 등

→ GPT-4가 맞춤 루틴 자동 생성!
```

### 3. 루틴 관리 📋
```
┌─────────────────────────┐
│ 💪 상체/하체 4주 루틴    │
│ 4주 · 주3회             │
│ ✅ 활성화됨             │
│ [편집] [복제] [삭제]    │
└─────────────────────────┘

기능:
├── 여러 루틴 저장 (최대 10개)
├── 1개만 활성화 가능
├── 운동 항목 수정
└── 루틴 복제/삭제
```

### 4. 캘린더 (메인 화면) 📅
```
[오늘] [주간] [월간]

오늘 뷰:
├── 오늘의 운동 To-do 리스트
├── 체크박스로 완료 표시
└── 진행률 표시

주간 뷰:
├── 월~금 운동 일정 그리드
├── 날짜 선택 시 운동 목록 표시
└── 완료/미완료 아이콘

월간 뷰:
├── 달력에 운동일 표시
├── ✅ 완료 / · 예정
└── 월간 통계
```

### 5. 진행 상황 추적 📊
- 이번 주 완료율
- 전체 루틴 진행률
- 연속 운동 일수
- 이번 달 기록

## 📱 화면 구성

### 유저 플로우

```
A. 첫 사용자:
로그인 → 프로필 설정 → 루틴 생성 → 캘린더

B. 기존 사용자:
로그인 → 캘린더 (메인)
```

### 하단 네비게이션 (4개 탭)

```
┌─────────────────────────────────┐
│  [마이]  [루틴]  [캘린더]  [진행] │
└─────────────────────────────────┘
```

| 탭 | 화면 | 설명 |
|----|------|------|
| 마이 | 마이페이지 | 프로필 설정 진입, 로그아웃 |
| 루틴 | 루틴 관리 | 루틴 목록, 활성화, 편집 |
| 캘린더 | 캘린더 | 오늘/주간/월간 뷰 (메인) |
| 진행 | 진행 상황 | 통계, 기록 |

## 🎬 사용자 시나리오

```
1️⃣ 구글 로그인
   └─ "Google로 계속하기" 클릭

2️⃣ 프로필 입력 (첫 사용자만)
   └─ 키, 몸무게, 목적, 체력 수준

3️⃣ 루틴 생성
   └─ 기간, 주당 횟수, 분할 방식 선택
   └─ 추가 요청: "어깨 부상이 있어요"
   └─ AI 루틴 생성 (3초)

4️⃣ 루틴 활성화
   └─ 생성된 루틴 자동 활성화
   └─ 다른 루틴 선택 시 전환

5️⃣ 캘린더에서 운동 관리
   └─ 오늘 뷰: To-do 체크
   └─ 주간 뷰: 일정 확인
   └─ 월간 뷰: 전체 현황

6️⃣ 진행 상황 확인
   └─ 이번 주 67% 완료
   └─ 3일 연속 운동 중!
```
