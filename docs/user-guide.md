# 사용자 가이드

## 📱 Plan4MyFit 사용법

Plan4MyFit에 오신 것을 환영합니다! 이 가이드는 앱의 모든 기능을 효과적으로 사용하는 방법을 안내합니다.

---

## 🚀 시작하기

### 1. 첫 로그인
1. **구글 로그인**: "Google로 계속하기" 버튼을 터치하세요
2. **구글 계정 선택**: 사용할 구글 계정을 선택하세요
3. **권한 승인**: 앱 사용에 필요한 권한을 승인하세요

### 2. 프로필 설정 (6단계)
첫 로그인 시 개인 맞춤 루틴 생성을 위한 정보를 입력합니다.

#### 1단계: 기본 정보
- **연령**: 15-80세 범위에서 입력
- **성별**: 남성, 여성, 기타 중 선택
- **키**: 100-250cm 범위에서 입력 (cm 단위)
- **몸무게**: 30-300kg 범위에서 입력 (kg 단위)

#### 2단계: 운동 환경
- **운동 장소**: 헬스장, 홈트레이닝, 야외운동, 복합 중 선택
- **주간 운동 횟수**: 1-7회 중 선택

#### 3단계: 목표 및 초점
- **운동 목표**: 근력 증가, 체중 감량, 체력 향상, 근육량 증가, 체형 교정 중 선택
- **운동 초점**: 상체 중심, 하체 중심, 전신 균형, 코어 강화 중 선택

#### 4단계: 신체 조건
- **체력 수준**: 입문자, 초급자, 중급자, 상급자 중 선택
- **불편한 부위**: 목, 어깨, 허리, 무릎, 발목, 손목, 없음 중 다중 선택 가능

#### 5단계: 운동 경력
- **경력 수준**: 없음, 6개월 미만, 6개월~1년, 1년~3년, 3년 이상 중 선택
- **상세 기록**: 경력이 있는 경우 주요 운동의 최대 중량 입력 (선택사항)

#### 6단계: 플랜 기간
- **플랜 기간**: 4주, 8주, 12주, 16주 중 선택

## 🤖 AI 루틴 생성

### 루틴 생성 과정
1. **루틴 탭** → **+ 버튼** 터치
2. **루틴 설정** 입력:
   - **기간**: 2-12주 선택
   - **주당 횟수**: 2-6회 선택
   - **분할 방식**: 전신, 상체/하체, 푸쉬/풀/레그 중 선택
   - **추가 요청**: 특별한 요청사항 입력 (선택사항)
3. **AI 루틴 생성** 버튼 터치
4. **로딩 대기** (30초-1분 소요)
5. **생성된 루틴 확인** 및 저장

### 분할 방식 가이드
- **전신**: 주 2-3회, 모든 근육을 한 번에 운동
- **상체/하체**: 주 3-4회, 상체와 하체를 번갈아 운동
- **푸쉬/풀/레그**: 주 5-6회, 밀기/당기기/다리 운동으로 분할

## 📋 루틴 관리

### 루틴 목록
- **루틴 카드**: 각 루틴의 이름, 기간, 분할 방식 표시
- **활성화 상태**: 현재 사용 중인 루틴에 "활성" 표시
- **생성일**: 루틴이 만들어진 날짜 표시

### 루틴 활성화
1. **루틴 목록**에서 사용할 루틴 선택
2. **활성화하기** 버튼 터치
3. 기존 활성 루틴은 자동으로 비활성화됨
4. **한 번에 1개 루틴만** 활성화 가능

### 루틴 편집
1. **루틴 카드**에서 **편집** 버튼 터치
2. **운동 목록** 확인
3. **운동 수정**:
   - 운동명, 세트 수, 반복 횟수 변경
   - 대상 근육 그룹 변경
4. **운동 추가**: + 버튼으로 새 운동 추가
5. **운동 삭제**: 휴지통 아이콘으로 운동 제거
6. **순서 변경**: 드래그 앤 드롭으로 운동 순서 조정

### 루틴 복제 및 삭제
- **복제**: 기존 루틴을 기반으로 새 루틴 생성
- **삭제**: 확인 후 루틴 완전 삭제 (복구 불가)
- **제한**: 최대 10개 루틴까지 저장 가능

## 📅 캘린더 사용법

### 오늘 뷰 (메인)
- **오늘의 운동**: 활성 루틴의 오늘 운동 목록 표시
- **체크박스**: 운동 완료 시 체크
- **진행률**: 완료된 운동 수 / 전체 운동 수
- **완료 메시지**: 모든 운동 완료 시 축하 메시지

### 주간 뷰
- **5일 그리드**: 월요일~금요일 운동 계획
- **상태 아이콘**: 
  - ✅ 완료
  - 📋 예정
  - 🔴 미완료
- **날짜 선택**: 특정 날짜 터치 시 해당 운동 목록 표시

### 월간 뷰
- **달력 형태**: 전체 월의 운동 기록
- **완료 표시**: 
  - ✅ 운동 완료일
  - · 운동 예정일
  - 빈칸 휴식일
- **월간 통계**: 하단에 완료율 및 총 운동 일수 표시

## 💡 사용 팁

### 효과적인 루틴 생성
1. **정확한 정보 입력**: 체력 수준과 경력을 정확히 입력하세요
2. **불편한 부위 표시**: 부상 방지를 위해 불편한 부위를 꼭 선택하세요
3. **현실적인 목표**: 주당 운동 횟수를 현실적으로 설정하세요
4. **추가 요청 활용**: 특별한 요구사항이 있다면 상세히 입력하세요

### 운동 기록 관리
1. **매일 체크**: 운동 후 바로 완료 체크하는 습관
2. **진행률 확인**: 주간 진행률을 정기적으로 확인
3. **연속 기록**: 연속 운동 일수를 늘려가며 동기 부여

### 루틴 편집 활용
1. **점진적 증가**: 체력 향상에 따라 세트나 반복 횟수 증가
2. **운동 교체**: 지루한 운동을 비슷한 효과의 다른 운동으로 교체
3. **순서 조정**: 개인 선호에 맞게 운동 순서 변경

## 🔧 문제 해결

### 자주 묻는 질문 (FAQ)

#### Q: 루틴 생성이 너무 오래 걸려요
A: AI 루틴 생성은 30초-1분 정도 소요됩니다. 네트워크 상태를 확인하고 잠시 기다려주세요.

#### Q: 운동 체크가 안 돼요
A: 페이지를 새로고침하거나 앱을 재시작해보세요. 문제가 지속되면 피드백을 보내주세요.

#### Q: 프로필을 수정하고 싶어요
A: 마이페이지 → 프로필 설정에서 언제든 수정 가능합니다.

#### Q: 루틴을 10개 이상 만들 수 없나요?
A: 현재 최대 10개까지 저장 가능합니다. 불필요한 루틴을 삭제 후 새로 생성하세요.

#### Q: 운동 동영상이나 설명이 있나요?
A: 현재는 운동명과 간단한 설명만 제공됩니다. 향후 업데이트에서 추가될 예정입니다.

### 기술적 문제
- **로딩이 안 될 때**: 브라우저 새로고침 (Ctrl+F5 또는 Cmd+R)
- **로그인 문제**: 브라우저 쿠키 및 캐시 삭제
- **화면 깨짐**: 브라우저 줌 레벨을 100%로 설정
