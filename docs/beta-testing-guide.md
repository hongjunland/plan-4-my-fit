# 베타 테스트 가이드

## 📋 베타 테스트 개요

Plan4MyFit 베타 테스트에 참여해주셔서 감사합니다. 이 가이드는 효과적인 테스트를 위한 지침을 제공합니다.

---

## 🎯 테스트 목표

### 주요 검증 사항
1. **사용자 경험 (UX)**: 직관적이고 사용하기 쉬운가?
2. **기능 완성도**: 모든 기능이 예상대로 작동하는가?
3. **성능**: 빠르고 안정적으로 동작하는가?
4. **모바일 최적화**: 모바일에서 편리하게 사용할 수 있는가?

---

## 👥 베타 테스터 프로필

### 타겟 그룹
- **헬스 초보자** (25-35세): 체계적인 운동 계획이 필요한 사용자
- **바쁜 직장인** (28-40세): 효율적인 운동 루틴을 원하는 사용자  
- **홈트족** (20-45세): 집에서 운동하는 사용자

### 테스터 모집 기준
- 스마트폰 사용에 익숙한 사용자
- 운동에 관심이 있는 사용자
- 피드백 제공에 적극적인 사용자

---

## 🧪 테스트 시나리오

### 시나리오 1: 첫 사용자 온보딩
```
1. 앱 접속 → 로그인 화면 확인
2. 구글 로그인 진행
3. 프로필 설정 (6단계) 완료
   - 기본 정보 (연령, 성별, 키, 몸무게)
   - 운동 환경 (장소, 주간 횟수)
   - 목표 및 초점 (목표, 초점 부위)
   - 신체 조건 (체력 수준, 불편한 부위)
   - 운동 경력 (경력 수준, 상세 기록)
   - 플랜 기간 설정
4. AI 루틴 생성 요청
5. 생성된 루틴 확인 및 활성화
6. 캘린더 화면으로 이동
```

### 시나리오 2: 일상적인 운동 관리
```
1. 캘린더 → 오늘 뷰에서 운동 목록 확인
2. 운동 항목 체크/언체크
3. 진행률 확인
4. 주간 뷰로 전환하여 주간 계획 확인
5. 월간 뷰로 전환하여 월간 통계 확인
```

### 시나리오 3: 루틴 관리
```
1. 루틴 탭으로 이동
2. 새 루틴 생성 (다른 설정으로)
3. 기존 루틴 편집 (운동 수정/추가/삭제)
4. 루틴 활성화 변경
5. 루틴 복제 및 삭제
```

### 시나리오 4: 진행 상황 추적
```
1. 진행 탭으로 이동
2. 주간/월간 완료율 확인
3. 연속 운동 일수 확인
4. 미니 캘린더에서 운동 기록 확인
```

## 📱 테스트 환경

### 지원 기기
- **iOS**: iPhone 12 이상, iOS 15+
- **Android**: Galaxy S21 이상, Android 11+
- **브라우저**: Chrome, Safari, Edge 최신 버전

### 테스트 조건
- **네트워크**: WiFi 및 4G/5G 환경
- **화면 크기**: 다양한 모바일 화면 크기
- **사용 시간**: 최소 30분 이상 사용

## 🐛 버그 리포팅 가이드

### 버그 발견 시 기록 사항
1. **재현 단계**: 버그가 발생한 정확한 단계
2. **예상 결과**: 어떤 결과를 예상했는지
3. **실제 결과**: 실제로 어떤 일이 발생했는지
4. **환경 정보**: 기기, 브라우저, 네트워크 상태
5. **스크린샷**: 가능한 경우 화면 캡처

### 버그 심각도 분류
- **Critical**: 앱 사용 불가능
- **High**: 주요 기능 동작 안함
- **Medium**: 일부 기능 이상
- **Low**: UI/UX 개선 사항
